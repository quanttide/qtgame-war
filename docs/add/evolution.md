# 可扩展性

## 加新战役

需要改的：
1. `assets/campaigns/{新战役名}/` 下三个 JSON（地图、兵种、战役参数）
2. 入口文件改一行指向新战役名
3. 胜利规则不一样的话加一个 `checkVictory` 变体

不需要改的：
- 模型定义（兵种、地形、战斗）
- 坐标计算（hex_toolkit）
- 战斗结算公式
