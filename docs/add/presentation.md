# 表现层

表现层画界面、收点击，不含规则。

## 地图渲染

每帧对每个格子依次画：

```
底色（地形颜色） → 地形图标（字符） → 高亮（蓝=可移动、红=可攻击、黄=选中） → 单位（圆+符号+血条+已行动遮罩）
```

额外的绘制：未暴露敌军画"?"。

## 点击处理

```
屏幕坐标 → 转格子坐标 → 判断格子类型
  ├── 己方 → 选中/切换
  ├── 敌方 → 攻击（如果在攻击范围内）
  ├── 空地 → 移动（如果在移动范围内）
  └── 长按 → 取消
```

## 信息面板

地图右侧的信息面板分几个独立区块：
- 单位列表：己方所有单位，点击可选中
- 战役面板：战力、工事强度、增援状态
- 详情面板：当前选中单位的属性和日志
