# 战棋游戏

## 动机

参考B站Up主”蘇北寒士“的视频合集”解放～你好，指挥官“。还原真实战场指挥官的视角做决策。

使用规则而非预设的故事，让玩家体验那种"明知道历史走向，却依然感到步步惊心、决策艰难"的压迫感。

## 玩法

从互动剧本模式开始逐渐熟悉兵棋推演模式。

### 指挥官：互动剧本模式

玩家在参谋部以指挥官视角决策：

1. 判断：在情报模糊、信息干扰等状态下筛选信息判断战场状况。
2. 命令：用选项 A、B、C 和自定义等下达，由 AI 参谋生成和后续执行。
3. 结算：是否完成战役战术目标、是否掌握战场主动权等。

### 作战科：兵棋推演模式 

玩家按照正常兵棋推演的玩法自定义地排兵部署。

## 建模

### 战场

- 格子：六边形单元格
- 棋盘：由符合真实战场比例的单元格构成
- 单位：作战单位、后勤单位等
- 玩家：人类、AI 等。
- 游戏：状态、结果等

### 指挥部

- 参谋部：情报、作战等。
- 政治部：动员、宣传等。
